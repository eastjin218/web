1.HTML 문법(태그)

->Hypertext Markup Language (HTML) is the standard markup language for creating web pages and web applications.
 1. 글자를 진하게 할떄 : strong
 2. 글자에 밑줄처리 할떄 : u
 3. 글자 제목 사용시 : h1~h6
 4. 글자 줄 변경 : br _닫기 태그필요없음
 5. 글자 줄 변경 : p _br과 차이는 html속에서 한 단락이라는 의미를 표현적으로 나타내기 때문
 6. 글자 목차 사용시 : li(list)
 7. li 태그의 부모태그인 ul(unordered list)태그가 있다 : li태그를 구별하고 그룹화할떄 ul사용
 8. ol(ordered list) 태그는 li의 부모태그로서 자동으로 목차의 넘버링을 할수 있다.
 9. table -> td -> tr 순으로 (전체->행 수 분류 -> 내용입력 및 열 분류)
10. a 태그 -> 웹에서 검색기능을 활용할수 있는 이유
   속성 : href= "주소" -> 주소링크, target="_blank" ->새창열기
